Тестовое задание

Цель:

Создать игру «Tower Defense» при помощи игрового движка Unity3D.
Описание:

В игре присутствует игровое поле, на нем расположены ячейки для установки башен. На
UI отображается счетчик валюты и 2 кнопки постройки башен:

- Лучник
- Подрывник
  
При клике на кнопку постройки башни, в случайном слоте строится соответствующая
башня и списывается валюта. Если валюты недостаточно, кнопка должна быть неактивной.

На игровом поле также есть дорожка, по которой идут враги. Появляются в начальной
точке и идут в конечную. При достижении финальной точки просто уничтожаются. Имеют
определенное количество здоровья.

Каждая башня должна иметь:
- Урон
- Радиус
- Скорость атаки
- Стоимость постройки
  
Башни должны атаковать противников, находящихся в радиусе и только после того, как
полностью повернулись в сторону противника. Поворот в сторону противника должен быть
плавным.

- Башня лучника: выстреливает снаряд, который летит в противника и наносит урон
- Башня подрывника: выстреливает снаряд, который наносит урон по области при
приземлении

Когда врага убивают (кончается здоровье), он уничтожается, за его убийство начисляется
валюта.

Визуал можно собирать из примитивных объектов.
